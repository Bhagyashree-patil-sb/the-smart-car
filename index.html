About The Project

This project is a Bluetooth Controlled Autonomous Smart Robot Car developed using Arduino UNO, ultrasonic sensors, servo motors, and MIT App Inventor.

The robot supports both manual and autonomous driving modes. Users can control the vehicle wirelessly using a mobile application. The ultrasonic sensor continuously detects obstacles while the servo motor scans safe directions for navigation.

Working of the Project

1. User sends commands through MIT App.
2. Bluetooth module receives commands.
3. Arduino UNO processes the instructions.
4. Motor driver controls movement.
5. Ultrasonic sensor detects obstacles.
6. Servo motor scans safe directions.
7. Robot automatically selects safest path.
